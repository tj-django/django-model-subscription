************
Installation
************

Via Python Package (Pip)
========================

.. code-block:: bash

    pip install django_model_subscription


Via Poetry
==========

.. code-block:: bash

    poetry add django_model_subscription

Via setup.py
============

Add to the project ``setup.py``.

.. code-block:: python

    setup(
        ...
        install_requires=['django_model_subscription'],
        ...
    )

Install your package in editable mode.

.. code-block:: bash

    pip install -e .
